# Slots

## Install

| type | command                        |
| ---- | ------------------------------ |
| npm  | `npm install @tocino-ui/slots` |
| yarn | `yarn add @tocino-ui/slots`    |
| pnpm | `pnpm add @tocino-ui/slots`    |

## Usage

```jsx
import Slots from `@tocino-ui/slots`;

function App() {
	return (
		<Slots />
	);
}
```
