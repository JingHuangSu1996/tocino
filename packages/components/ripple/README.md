# Ripple

## Install

| type | command                         |
| ---- | ------------------------------- |
| npm  | `npm install @tocino-ui/ripple` |
| yarn | `yarn add @tocino-ui/ripple`    |
| pnpm | `pnpm add @tocino-ui/ripple`    |

## Usage

```jsx
import Ripple from `@tocino-ui/ripple`;

function App() {
	return (
		<Ripple />
	);
}
```
