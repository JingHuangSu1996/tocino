# FocusScope

## Install

| type | command                              |
| ---- | ------------------------------------ |
| npm  | `npm install @tocino-ui/focus-scope` |
| yarn | `yarn add @tocino-ui/focus-scope`    |
| pnpm | `pnpm add @tocino-ui/focus-scope`    |

## Usage

```jsx
import FocusScope from `@tocino-ui/focus-scope`;

function App() {
	return (
		<FocusScope />
	);
}
```
