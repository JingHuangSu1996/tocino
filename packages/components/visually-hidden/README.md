# VisuallyHidden

## Install

| type | command                                  |
| ---- | ---------------------------------------- |
| npm  | `npm install @tocino-ui/visually-hidden` |
| yarn | `yarn add @tocino-ui/visually-hidden`    |
| pnpm | `pnpm add @tocino-ui/visually-hidden`    |

## Usage

```jsx
import VisuallyHidden from `@tocino-ui/visually-hidden`;

function App() {
	return (
		<VisuallyHidden />
	);
}
```
