# Utils

## Install

| type | command                        |
| ---- | ------------------------------ |
| npm  | `npm install @tocino-ui/utils` |
| yarn | `yarn add @tocino-ui/utils`    |
| pnpm | `pnpm add @tocino-ui/utils`    |

## Usage

```jsx
import Utils from `@tocino-ui/utils`;

function App() {
	return (
		<Utils />
	);
}
```
