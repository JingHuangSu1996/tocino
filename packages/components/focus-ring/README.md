# FocusRing

## Install

| type | command                             |
| ---- | ----------------------------------- |
| npm  | `npm install @tocino-ui/focus-ring` |
| yarn | `yarn add @tocino-ui/focus-ring`    |
| pnpm | `pnpm add @tocino-ui/focus-ring`    |

## Usage

```jsx
import FocusRing from `@tocino-ui/focus-ring`;

function App() {
	return (
		<FocusRing />
	);
}
```
