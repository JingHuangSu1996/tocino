# Button

## Install

| type | command                         |
| ---- | ------------------------------- |
| npm  | `npm install @tocino-ui/button` |
| yarn | `yarn add @tocino-ui/button`    |
| pnpm | `pnpm add @tocino-ui/button`    |

## Usage

```jsx
import Button from `@tocino-ui/button`;

function App() {
	return (
		<Button />
	);
}
```
